# 简易学生成绩管理系统

## 页面介绍
### 搜索界面
我们可以搜索学生、老师、课程、班级的信息。选择对应的类型，搜索即可

<image src="https://user-images.githubusercontent.com/78332649/215435371-6c7368b3-bb52-4067-aea6-0eda3443dc39.png" width="80%"/>
 
- 关于学生和他/她选择的课程的信息，课程的分数，学生的平均分数，所有学生都可以根据学生的ID或姓名进行查询。如果没有给出ID和姓名，显示所有学生及其课程信息。
![image](https://user-images.githubusercontent.com/78332649/216061128-977b3d9d-999f-4232-bfde-5fb27bf7c050.png)

- 可以基于教师的姓名或ID来查询关于教师或教师所教课程的信息。如果没有给出教师的姓名或ID，则显示所有教师或他们教授的所有课程的信息。
![image](https://user-images.githubusercontent.com/78332649/216061568-fb24252e-bdbd-402a-b42d-f4eb6fc32e4b.png)

- 可以根据课程名称或ID查询课程或选课、课程分数、课程平均分数等信息。如果没有给出课程名称或ID，则显示所有课程或所有选课的信息。 
![image](https://user-images.githubusercontent.com/78332649/216062143-0227c344-6c3b-4906-8659-d01c336d0c7a.png)

- 可以根据班级名称或ID查询学生信息、同一班级学生的平均成绩。如果没有给出类的名称，显示所有类的信息。 
![image](https://user-images.githubusercontent.com/78332649/215435688-091ca1da-8a84-4241-887b-9ec28fda8e11.png)
